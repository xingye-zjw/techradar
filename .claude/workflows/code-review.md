# 代码审查工作流

## 审查维度

### 1. 代码质量审查
- TypeScript 类型安全性
- 代码风格一致性
- 最佳实践遵循
- 错误处理机制

### 2. 性能审查
- 渲染性能优化
- 内存使用情况
- 构建产物大小
- 加载性能

### 3. 安全审查
- 敏感信息暴露
- XSS 漏洞风险
- 依赖安全
- 配置安全

### 4. 架构审查
- 模块化设计
- 代码复用性
- 可维护性
- 扩展性

## 审查文件范围

- `app/**/*.tsx` - 页面组件
- `components/**/*.tsx` - UI 组件
- `lib/**/*.ts` - 业务逻辑
- `next.config.js` - 配置文件
- `package.json` - 依赖配置

## 输出格式

每个维度输出：
1. 发现的问题列表（按严重程度：高/中/低）
2. 具体代码位置和行号
3. 修复建议
4. 最佳实践示例
