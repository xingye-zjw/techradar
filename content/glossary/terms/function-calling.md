# 函数调用

Function Calling（也称 Tool Use）是 LLM Agent 系统的基础能力：模型在回复中插入结构化的工具调用请求（如 get_weather(location='北京')），由外部程序执行后把结果传回模型继续处理。实现方式是在 Chat Completion API 中声明可用工具的 schema（名称、参数类型、描述），模型会根据对话内容决定是否调用以及传入什么参数。这解决了 LLM 的两大局限：知识截止日期限制和无法执行计算/查询的动态需求。
