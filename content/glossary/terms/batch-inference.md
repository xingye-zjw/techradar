# 批量推理

批量推理是在 GPU 上同时处理多个推理请求以提升吞吐量的技术。相比在线推理（一次处理一个请求），批量推理通过矩阵运算充分调动 GPU 并行能力，QPS 可提升 10-100x。连续批处理（Continuous Batching / Iteration-Level Scheduling）在每个 token 生成后动态插入新请求，不等当前 batch 全部完成就开始处理下一批，是 vLLM 等推理引擎的核心优化。
