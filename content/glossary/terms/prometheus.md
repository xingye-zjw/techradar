# Prometheus

Prometheus 是 CNCF 的开源时序数据库和监控系统，采用 Pull 模式从每个服务的 /metrics 端点定期拉取数据。核心概念：四大指标类型——Counter（只增计数器）、Gauge（可增可减仪表）、Histogram（延迟分布直方图，用于计算 P50/P95/P99）、Summary（预计算分位数）；PromQL 是其查询语言，支持 rate()、increase()、histogram_quantile() 等时间序列函数。Alertmanager 接收 Prometheus 告警并路由到邮件/Slack 等渠道。
