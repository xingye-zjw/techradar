## 为什么你要学它

PCB（Printed Circuit Board，印制电路板）是电子产品的物理载体。在硬件开发中：

- **从原理图到实物**：原理图只是逻辑连接，PCB才是可生产的物理设计
- **信号完整性**：高速信号需要正确的阻抗匹配、差分走线、等长匹配
- **EMI/EMC合规**：产品必须通过电磁兼容测试才能上市
- **成本控制**：合理的PCB设计能减少层数、缩小面积、提高良率
- **可靠性**：热设计、电源完整性、DFM（可制造性设计）决定产品寿命

如果你只会画原理图而不会PCB设计，你的设计永远停留在纸面上，无法变成真正的产品。

## 一句话概览（快速版）

- **PCB是电子元件的家**：提供机械支撑和电气连接
- **层叠结构是基础**：信号层、电源层、地层合理分配
- **布局决定一半成败**：先放核心器件，再按信号流向布局
- **布线遵循规则**：3W原则、20H原则、差分等长、阻抗匹配
- **DRC是最后一道关**：设计规则检查确保无短路、开路、间距违规

```python
# 层叠厚度计算（阻抗控制）
def microstrip_impedance(h, w, t, er):
    """
    微带线阻抗计算（简化公式）
    h: 介质厚度（mil）
    w: 走线宽度（mil）
    t: 铜厚（mil）
    er: 介电常数（FR4约4.5）
    """
    # 简化公式（实际用专业软件计算）
    z0 = 87 / np.sqrt(er + 1.41) * np.log(5.98 * h / (0.8 * w + t))
    return z0

def stripline_impedance(h, w, er):
    """
    带状线阻抗计算（简化公式）
    h: 走线到参考平面距离
    w: 走线宽度
    er: 介电常数
    """
    z0 = 60 / np.sqrt(er) * np.log(4 * h / (0.67 * w))
    return z0

# 示例：计算50Ω微带线
import numpy as np
h = 5  # 5mil介质厚度
er = 4.5  # FR4
# 调整w使Z0≈50Ω
w = 10  # 约10mil宽度
z0 = microstrip_impedance(h, w, 1.4, er)
print(f"微带线阻抗: {z0:.1f}Ω")
```

### 🔑 PCB布线规则

```python
# 布线核心原则

# 1. 走线宽度与电流
def trace_width_current(current, temp_rise=10, copper_weight=1):
    """
    计算走线宽度（IPC-2221标准简化）
    current: 电流（A）
    temp_rise: 允许温升（°C）
    copper_weight: 铜厚（oz）
    返回：走线宽度（mil）
    """
    # 简化公式
    k = 0.048  # 外层系数
    width = k * (current ** 1.37) * (temp_rise ** 0.21) * copper_weight
    return width

# 常用走线宽度
trace_widths = {
    "信号线": "6-10mil",
    "电源线": "20-50mil（根据电流）",
    "地线": "尽量宽",
    "大电流": "≥100mil或使用铜箔",
}

# 2. 间距规则
clearance_rules = {
    "一般信号": "6-8mil",
    "电源到信号": "≥10mil",
    "高压（>50V）": "≥20mil",
    "差分对内": "根据阻抗计算",
    "差分对间": "≥3倍对内间距",
}

# 3. 过孔规则
via_rules = {
    "标准过孔": "外径20mil，内径12mil",
    "小过孔": "外径12mil，内径6mil",
    "盲孔/埋孔": "HDI工艺，成本高",
    "过孔到焊盘": "≥8mil",
    "电源过孔": "多打几个减小阻抗",
}

# 4. 差分走线
differential_rules = """
差分走线要点：
- 等长：误差控制在±5mil以内
- 等距：保持固定间距
- 阻抗：通常90Ω或100Ω差分阻抗
- 同层：尽量在同一层走线
- 最少过孔：减少阻抗不连续
"""

# 差分阻抗计算
def differential_impedance(z0, s, h):
    """
    差分阻抗计算（简化）
    z0: 单端阻抗
    s: 差分线间距
    h: 到参考平面距离
    """
    z_diff = 2 * z0 * (1 - 0.48 * np.exp(-0.96 * s / h))
    return z_diff

# 5. 3W原则和20H原则
design_rules = """
3W原则：
- 走线中心间距≥3倍线宽
- 减少串扰，提高信号完整性

20H原则：
- 电源平面比地平面内缩20H
- H为层间距
- 减少边缘辐射，改善EMI
"""

# 6. 回流路径
return_path = """
关键原则：信号走哪里，回流就走哪里

- 高频信号回流沿阻抗最低路径（正下方）
- 换层时必须打地过孔提供回流路径
- 跨分割会导致回流路径增大，EMI恶化
- 地平面完整是信号完整性的基础
"""
```

```python
    er = 4.5  # 介电常数
    h = 5  # 层间距（mil）
    t = 1.4  # 铜厚（mil）
    
    # 调整w和s使Zdiff ≈ 90Ω
    w = 8  # 线宽（mil）
    s = 6  # 间距（mil）
    
    # 单端阻抗
    z0 = 87 / np.sqrt(er + 1.41) * np.log(5.98 * h / (0.8 * w + t))
    
    # 差分阻抗
    z_diff = 2 * z0 * (1 - 0.48 * np.exp(-0.96 * s / h))
    
    return z0, z_diff

z0, z_diff = usb_differential_trace()
print(f"单端阻抗: {z0:.1f}Ω")
print(f"差分阻抗: {z_diff:.1f}Ω")
```

## 常见误区

**误区 1：地平面随便分割 → 信号回流路径混乱**

解释：地平面是信号的回流路径，随意分割会导致回流绕路，产生EMI问题。模拟地和数字地可以分割，但必须单点连接，且连接点要仔细选择。

**误区 2：去耦电容随便放 → 滤波效果大打折扣**

解释：去耦电容必须紧贴芯片电源引脚，距离越近越好。电容的等效串联电感（ESL）和等效串联电阻（ESR）会随距离增加而恶化，导致高频滤波失效。

**误区 3：走线越短越好 → 忽略等长匹配**

解释：对于时序敏感的信号（如DDR、PCIe），等长比最短更重要。差分对、地址数据线都需要等长匹配，否则会产生时序问题。

**误区 4：过孔随便打 → 阻抗不连续**

解释：过孔会产生阻抗不连续，高速信号应尽量少换层。必须换层时，要在信号过孔旁边打地过孔，提供回流路径。

**误区 5：只看原理图不看数据手册 → 封装错误**

解释：同一芯片可能有多种封装（QFP、QFN、BGA），引脚定义可能不同。设计前必须核对数据手册的封装尺寸和引脚定义。
