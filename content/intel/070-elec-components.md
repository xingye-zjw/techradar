## 为什么你要学它

先讲结论：**不懂电子元件，你连电路图都看不懂，更别说DIY任何电子作品。**

无论是你手机里的芯片、宿舍的LED灯、还是实验室的传感器接口，底层都是这些基本元件的排列组合。电阻、电容、电感是电路的"建筑砖块"；二极管、三极管、MOS管是控制电流的"阀门"；运放则是电路中的"放大器"。不理解它们，你看任何硬件文档都会一头雾水。

电子技术是软硬结合的交汇点。学了它，你才能真正理解传感器数据从何而来、电机怎么控制、以及为什么嵌入式开发必须懂点硬件。

## 一句话概览（快速版）

你只要记住三句话：

1. **电阻限流分压、电容隔直通交储能、电感隔交通直储能**——三大被动元件各有分工
2. **二极管单向导通、三极管/MOS管可控开关**——半导体元件是电路的控制核心
3. **运放 = 集成化的放大器**——用少量元件实现精确的信号处理

## 核心拆解

### 🔑 电阻（Resistor）

**核心特性**：限制电流、分压

- **欧姆定律**：V = I × R，理解电压、电流、电阻三者关系
- **色环读取**：黑棕红橙黄绿蓝紫灰白分别对应0-9，乘以10^n
- **功率**：1/4W最常用，大功率需要更大封装

```python
# 分压公式
V_out = V_in * R2 / (R1 + R2)

# 限流计算
I = (V_supply - V_led) / R_led
```

**典型应用**：LED限流电阻、上拉/下拉电阻、偏置电阻

### 🔑 电容（Capacitor）

**核心特性**：隔直通交、储能滤波

- **容抗**：Xc = 1 / (2πfC) —— 频率越高，容抗越低
- **储能公式**：E = ½CV²
- **滤波原理**：电容为交流提供低阻抗通路，滤除纹波

```python
# RC滤波截止频率
f_c = 1 / (2 * 3.14159 * R * C)

# 例如：R=1kΩ, C=100μF → f_c ≈ 1.6Hz（低通滤波）
```

**典型应用**：电源滤波、去耦电容、隔直电容、RC定时

### 🔑 电感（Inductor）

**核心特性**：隔交通直、储能

- **感抗**：XL = 2πfL —— 频率越高，感抗越高
- **储能公式**：E = ½LI²
- **与电容对偶**：电容阻高频、通低频；电感阻低频、通高频

**典型应用**：开关电源的储能电感、EMI滤波、谐振电路

### 🔑 二极管（Diode）

**核心特性**：单向导通

- **正向压降**：硅管约0.7V，锗管约0.3V，LED约1.8-3.3V
- **反向击穿**：齐纳二极管利用反向击穿特性做稳压
- **常用类型**：整流二极管、肖特基二极管（低压降）、齐纳二极管（稳压）、TVS二极管（瞬态保护）

```python
# 二极管钳位电路
# 当输入电压 > 0.7V 时，二极管导通，输出被钳位在 0.7V
V_out = min(V_in, V_diode_on)  # 简化模型
```

### 🔑 三极管（BJT）与MOS管

**三极管（BJT）**：电流控制型
- **工作区域**：放大区（发射结正偏、集电结反偏）、截止区、饱和区
- **电流关系**：Ie = Ib + Ic，Ic = β × Ib（β为电流增益）
- **应用**：小信号放大、作为开关驱动LED/继电器

**MOS管（MOSFET）**：电压控制型
- **栅极电阻极大**，几乎不取电流——比三极管更适合作为数字开关
- **N沟道 vs P沟道**：N沟道常用作低端开关（接地端），P沟道常用作高端开关（电源端）
- **导通条件**：N沟道 Vgs > Vth（通常2-4V），P沟道 Vgs < -Vth

```python
# Arduino控制MOS管示例
# IRF540N N沟道MOS管，Vgs(th) ≈ 4V
if digital_write(HIGH):  # Arduino输出5V
    MOSFET导通，负载得电
else:
    MOSFET截止，负载失电
```

### 🔑 运算放大器（Op-Amp）

**核心公式**：虚短（V+ ≈ V-）、虚断（输入电流 ≈ 0）

- **反相放大**：Vout = - (R2/R1) × Vin
- **同相放大**：Vout = (1 + R2/R1) × Vin
- **电压跟随器**：Vout = Vin（缓冲/阻抗变换）
- **比较器**：V+ > V- → 输出高；V+ < V- → 输出低

### 🔑 读懂原理图

**基本原则**：
- **信号流向**：从左到右，从上到下
- **电源符号**：VCC/VDD为正， GND/VSS为负/地
- **节点**：线条交叉有点表示连接，无点表示跨线
- **去耦电容**：电源引脚附近接0.1μF电容到地，滤除高频噪声

**常用符号认知**：
```
电阻：矩形 或 锯齿线
电容：两条平行横线（有极性电容有一条弧线）
电感：螺旋线
二极管：三角+竖线（箭头指向阴极）
三极管：圆形+三个引脚（b/c/e）
运放：三角形（输入在左，输出在右）
```

## 完整跑通方案

### 实验一：电阻分压实测

**目标**：用万用表测量分压比例，验证欧姆定律

**器材**：9V电池、10kΩ电阻、4.7kΩ电阻、面包板、万用表

**步骤**：
1. 按 R1=10kΩ、R2=4.7kΩ 连接分压电路
2. 测量输入电压 Vin = 9V
3. 测量输出电压 Vout = Vin × R2/(R1+R2) ≈ 2.9V
4. 用万用表实测，对比理论计算

**预期结果**：实测值与理论值误差 < 5%

### 实验二：电容滤波观测

**目标**：用示波器观察电容的滤波效果

**器材**：Arduino/信号发生器、10μF电解电容、0.1μF陶瓷电容、1kΩ电阻、示波器

**步骤**：
1. Arduino输出50%占空比PWM（频率约490Hz）
2. 不接电容，用示波器观测波形（有较大纹波）
3. 并联10μF电容，观测波形（纹波减小）
4. 再并联0.1μF电容，观测高频噪声被进一步滤除

**预期结果**：电容越大、纹波越小；陶瓷电容滤高频、电解电容滤低频

### 实验三：晶体管开关驱动LED

**目标**：用三极管作为开关，由Arduino控制LED

**器材**：Arduino、NPN三极管（如2N2222）、LED、220Ω电阻、1kΩ电阻

**电路**：
```
Arduino Pin 9 --- 1kΩ ---+--- B（基极）
                          |
                       2N2222
                          |
GND ----------------------+--- E（发射极）
                          |
                         LED
                          |
                       220Ω
                          |
                        VCC(5V)
```

**代码**：
```python
# Arduino代码
void setup() {
    pinMode(9, OUTPUT);
}

void loop() {
    digitalWrite(9, HIGH);  # 三极管导通，LED亮
    delay(1000);
    digitalWrite(9, LOW);  # 三极管截止，LED灭
    delay(1000);
}
```

**预期结果**：LED以1Hz频率闪烁

### 实验四：运放反相放大器

**目标**：用运放搭建一个放大10倍的电路

**器材**：LM358运放、±9V电源（或电荷泵芯片）、10kΩ电阻×2、100kΩ电阻×1、信号源、示波器

**电路**：
```
信号源 --- 10kΩ ---+--- 运放负输入端（V-）
                   |
                   100kΩ（反馈电阻）
                   |
                   +--- 运放输出端（Vout）
                   
运放正输入端（V+）--- GND
```

**公式**：Vout = - (100k/10k) × Vin = -10 × Vin

**步骤**：
1. 输入1kHz正弦波，幅度100mV
2. 用示波器观测输出，应该反相且幅度约1V
3. 改变输入频率，观察增益是否保持（运放带宽积有限）

## 常见误区

**"电容容量越大越好" → 错**。容量大成本高、体积大、漏电也大。滤波电容选型要综合考虑纹波电流、ESR、工作电压。电源去耦通常用一大（10μF）一小（0.1μF）组合。

**"三极管和MOS管可以混用" → 错**。三极管是电流驱动，需要基极电流；MOS管是电压驱动，栅极几乎不取电流。选型时要确认驱动电路能力。MOS管GS极间有结电容，开关速度受充放电时间限制。

**"运放输出可以无限放大" → 错**。运放输出受电源电压限制，且输出电流有限（通常几十mA）。另外增益带宽积（GBW）是常数，高增益时带宽会下降。

**"原理图中相邻的电容一定是去耦电容" → 不一定**。要结合位置判断：电源引脚附近的是去耦，信号链路中的是耦合/滤波。用错了可能引入噪声或阻断直流信号。

**"LED不用限流电阻直接接5V" → 烧毁**。LED的VF通常2-3V，若直接接5V，电流可达上百mA，会立即烧毁。即使短时间亮了，寿命也极短。
