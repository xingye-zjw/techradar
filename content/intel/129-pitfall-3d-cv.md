## 点云数据预处理不当导致信息丢失

// 快速修复

归一化到单位球 + FPS下采样 + 保持密度一致

// 现象表现

- × 同一物体不同视角识别差异大
- × 点云稀疏部分特征丢失
- × 模型泛化能力差

// 排查步骤

- 01 检查点云归一化方式，确认是否统一到单位球或单位立方体
- 02 用FPS（最远点采样）保证点云均匀分布，避免密度差异过大
- 03 增加数据增强（旋转、平移、缩放、加噪）提升模型泛化能力

#点云#预处理#数据增强

## ICP配准陷入局部最优

// 快速修复

多初始值尝试 + 粗配准 + 使用RANSAC先对齐

// 现象表现

- × ICP配准结果偏差很大
- × 每次运行结果不一样
- × 收敛到错误的对齐位置

// 排查步骤

- 01 先用RANSAC或特征匹配（如FPFH）做粗配准，提供良好初始姿态
- 02 多初始姿态取最优结果，避免单一初始值陷入局部最优
- 03 检查对应点对的正确性，剔除离群点，设置合理的距离阈值

#ICP#配准#局部最优

## 体素分辨率与显存的矛盾

// 快速修复

稀疏体素 + 多尺度体素 + 八叉树表示

// 现象表现

- × 高分辨率体素显存不足
- × 低分辨率精度不够
- × GPU OOM（内存溢出）

// 排查步骤

- 01 使用稀疏卷积（SPConv）替代密集体素卷积，大幅减少显存占用
- 02 采用八叉树（Octree）表示替代密集体素，只保留有效空间
- 03 降采样输入点云或直接使用点云网络（如PointNet系列）替代体素方案

#体素#显存#3D检测

## 相机标定误差导致3D重建不准

// 快速修复

重新标定 + 多视角校验 + 使用标定板

// 现象表现

- × 重建结果有明显畸变
- × 深度估计偏差大
- × 不同视角点云无法对齐

// 排查步骤

- 01 检查内参标定质量，使用张正友标定法重新采集标定板图像
- 02 验证外参准确性，多视角交叉校验相机位姿
- 03 使用标准标定物（如棋盘格、标定球）评估重建精度

#3D重建#相机标定#精度
