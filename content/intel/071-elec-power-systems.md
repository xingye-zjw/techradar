## 为什么你要学它

先讲结论：**电力系统是所有硬科技基础设施的底层。** 不懂它，你看新能源汽车、光伏逆变器、数据中心供电、工业电机控制，都是浮云。

具体来说：
- **新能源并网**：光伏/风电产生的电要送入电网，需要理解功率平衡、电压频率控制
- **电力电子**：所有功率变换器（AC/DC、DC/AC、升压/降压）的本质是**阻抗变换 + 能量路由**
- **系统稳定性**：为什么电网不会轻易崩溃？频率-功率 droop 特性是怎么回事？
- **硬件选型**：选电缆截面、确定变压器容量、计算温升，都需要电力系统基础知识

如果你是做硬件（功率半导体、电机驱动）、能源互联网、电力市场，电力系统是绕不过去的坎。

## 一句话概览（快速版）

你只要记住三句话：

1. **电力系统的核心矛盾是"电能不能大规模存储"** → 发电必须 = 用电，实时平衡
2. **交流电网（AC）统治世界的原因是：变压器能轻松改变电压等级**，实现高效输电
3. **三相电是最优的输电方式**——三根线比单相更高效、电机启动更平稳、功率更恒定

## 核心拆解

### 🔑 交流电（AC）vs 直流电（DC）

**交流电（AC）**：
- 电压/电流随时间做正弦变化：`v(t) = V_m sin(ωt)`
- 频率：50Hz（中国/欧洲）或 60Hz（美国/日本）
- **优势**：电压可通过变压器轻松变换，发电/输电/配电系统结构简单
- **劣势**：存在集肤效应、谐波问题、无功功率

**直流电（DC）**：
- 电压恒定或纹波很小
- **优势**：无无功功率、适合远距离海缆输电（HVDC）、便于储能系统接口
- **劣势**：电压变换需要电力电子变换器，成本高

**为什么交流赢了历史**：20世纪初的"电流战争"（爱迪生 vs 特斯拉），特斯拉的交流系统在长距离输电上胜出，因为变压器让高压输电成为可能（`P = V × I`，电压越高，电流越小，损耗 `I²R` 越小）。

### 🔑 三相电（Three-Phase）

三相电是电力系统的标准制式，由三个相位相差 120° 的单相组成：

```
V_A = V_m sin(ωt)
V_B = V_m sin(ωt - 120°)
V_C = V_m sin(ωt + 120°)
```

**三相电的优势**：

| 特性 | 单相 | 三相 |
|------|------|------|
| 导线材料 | 2根 | 3根（星形）或 4根 |
| 功率传输 | 脉动（有零点） | 恒定（无脉动） |
| 电机启动 | 需要辅助绕组 | 直接启动 |
| 输电效率 | 较低 | 高（芯子利用率高） |

**功率计算**（三相平衡）：
```python
# 视在功率 S = √3 × V_L × I_L
# 有功功率 P = √3 × V_L × I_L × cos(φ)
# 无功功率 Q = √3 × V_L × I_L × sin(φ)
# 功率因数 cos(φ) = P / S

V_L = 380  # 线电压 (V)
I_L = 100  # 线电流 (A)
cos_phi = 0.85  # 功率因数

S = 1.732 * V_L * I_L  # 视在功率 VA
P = S * cos_phi  # 有功功率 W
Q = S * 0.619  # sin(arccos(0.85)) ≈ 0.527

print(f"S = {S:.0f} VA")  # S = 65816 VA ≈ 65.8 kVA
print(f"P = {P:.0f} W")   # P = 55944 W ≈ 55.9 kW
```

### 🔑 变压器原理

变压器基于**电磁感应**，实现电压/电流的变换，同时保持功率近似守恒。

**理想变压器方程**：
```
V1 / V2 = N1 / N2 = a（变比）
I1 / I2 = N2 / N1 = 1/a
S1 ≈ S2（P 基本守恒）
```

**实际变压器的等效电路**：
```
         R1     X1               R2'    X2'
---[///]--+--[////]--+     +--[///]--+--[////]---+---
         |          |     |          |           |
      [励磁支路]    +-----+     [负载]            |
         |          |     |          |           |
------------------+----------+--------------------+----
                 
R1, X1：原边阻抗（铜损+漏抗）
R2', X2'：副边折算到原边的阻抗
励磁支路：铁损（Rc）+ 励磁电抗（Xm）
```

**关键参数**：
- **额定容量**（kVA）：变压器能传输的最大视在功率
- **短路阻抗**（% Z）：短路时的等效阻抗，决定短路电流大小
- **空载电流**（% I0）：原边加额定电压副边开路时的电流（通常 0.5-2%）

### 🔑 电网结构

电力系统按电压等级分层：

| 层级 | 电压等级 | 功能 |
|------|----------|------|
| 输电网 | 220kV / 500kV / 750kV | 大容量远距离输电，互联区域电网 |
| 配电网 | 10kV / 35kV / 110kV | 分配电能到用户侧 |
| 用电侧 | 220V / 380V | 家庭/工业用电 |

**电网调度核心约束**：
1. **功率平衡**：发电 = 用电 + 损耗（频率是平衡的天然指标）
2. **电压约束**：每点电压必须在额定 ±5~10% 内
3. **热极限约束**：线路电流不能超过热稳定电流

### 🔑 输电线路模型

输电线路根据长度采用不同等效模型：

**短线路（< 80km）**：只考虑串联阻抗
```
Z = R + jX = (r + jx) × L
ΔU = I × Z（电压损耗）
```

**中等线路（80-250km）**：考虑对地电容
**长线路（> 250km）**：采用分布参数模型

**π 型等效电路**（常用）：
```
          Z/2         Z/2
---[///]--+---+---[///]--+---+---
         |   |         |   |
        [C] [C]       [C] [C]
         |   |         |   |
---------+---+---------+---+-------
```

**线路损耗计算**：
```python
# 三相线路损耗
# P_loss = 3 × I² × R（热损耗）
# 电压损耗 ΔU% = (P × L × r + Q × L × x) / (U² × 10)

# 示例：10km 380V 三相线路，传输 100kW，功率因数 0.85
P = 100_000  # W
U = 380      # V
L = 10       # km
r = 0.27     # Ω/km（铜芯电线约 0.27 Ω/km for 50mm²）
R = r * L    # 总电阻

I = P / (1.732 * U * 0.85)  # 线电流
P_loss = 3 * I**2 * R        # 损耗功率

print(f"I = {I:.1f} A")           # I = 178.6 A
print(f"P_loss = {P_loss:.0f} W") # P_loss = 2586 W ≈ 2.6 kW
print(f"损耗率 = {P_loss/P*100:.2f}%")  # 损耗率 = 2.59%
```

## 完整跑通方案

### 第一步：理解三相电功率计算

```python
import numpy as np

def three_phase_power(V_LL, I_L, cos_phi):
    """计算三相电功率"""
    S = np.sqrt(3) * V_LL * I_L      # 视在功率 VA
    P = S * cos_phi                    # 有功功率 W
    Q = S * np.sin(np.arccos(cos_phi)) # 无功功率 var
    return {'S_VA': S, 'P_W': P, 'Q_var': Q, 'cos_phi': cos_phi}

# 例：某工厂进线 380V，电流 200A，功率因数 0.9
result = three_phase_power(380, 200, 0.9)
print(f"视在功率: {result['S_VA']/1000:.1f} kVA")
print(f"有功功率: {result['P_W']/1000:.1f} kW")
print(f"无功功率: {result['Q_var']/1000:.1f} kvar")
# 输出：
# 视在功率: 131.5 kVA
# 有功功率: 118.4 kW
# 无功功率: 57.3 kvar
```

### 第二步：潮流计算（Load Flow）—— 简单迭代法

**目标**：已知电网结构，求解各节点电压和功率分布。

```python
def simple_load_flow(V_source, Z_line, P_load, Q_load, tolerance=1e-6, max_iter=100):
    """
    简化潮流计算：已知首端电压、线路阻抗、负载功率
    求解末端电压和功率分布
    """
    V = V_source  # 初始猜测
    
    for i in range(max_iter):
        # 计算线路电流（S = V²/Y*...简化用 P = V²/R）
        # 这里用简化的迭代：ΔV = (P*R + Q*X) / V
        delta_V = (P_load * Z_line.real + Q_load * Z_line.imag) / V
        
        V_new = V - delta_V
        
        if abs(V_new - V) < tolerance:
            print(f"收敛于第 {i+1} 次迭代，末端电压 {V_new:.2f} V")
            return V_new
        
        V = V_new
    
    print(f"达到最大迭代次数 {max_iter}，末端电压 {V:.2f} V")
    return V

# 示例：220V 电源，经阻抗 Z=0.5+j0.8 Ω 的线路，供给 10kW @ 0.85pf 负载
V_source = 220
Z_line = 0.5 + 1j * 0.3
P_load = 10_000
Q_load = 10_000 * np.tan(np.arccos(0.85))

V_end = simple_load_flow(V_source, Z_line, P_load, Q_load)
# 电压损耗
print(f"电压损耗: {V_source - V_end:.2f} V ({(V_source-V_end)/V_source*100:.2f}%)")
```

### 第三步：短路故障分析

**对称短路（三相短路）**电流计算：

```python
def three_phase_fault_current(V_base, Z_source, Z_line):
    """
    计算三相短路电流
    I_fault = V_base / (Z_source + Z_line)
    """
    Z_total = Z_source + Z_line
    I_fault = V_base / Z_total
    return abs(I_fault)

# 示例：变压器短路阻抗 5%，系统阻抗忽略，基准电压 10kV
V_base = 10_000  # V
Z_transformer = 0.05  # 5% = j0.05 pu（折算后）
Z_line = 0.2 + 1j * 0.5  # Ω

I_fault = three_phase_fault_current(V_base, Z_transformer, Z_line)
print(f"三相短路电流: {I_fault:.0f} A")  # ≈ 16500 A

# 断路器选型：短路电流是额定电流的 5-10 倍，需要选用 Icu > 20kA 的断路器
```

### 第四步：实际案例 —— 光伏并网对配电网的影响

1. **问题**：光伏逆变器并网，功率倒送导致电压升高
2. **分析**：用潮流计算看不同渗透率下的节点电压

## 常见误区

**"380V 和 220V 是什么关系？" → 380V 是线电压（任意两相之间），220V 是相电压（任意一相对中性点）**。在星形接法中，`V_L = √3 × V_Ph`。所以 380V ÷ √3 ≈ 220V。

**"功率因数越高越好？" → 不完全是**。功率因数过低，无功功率大，线路电流大，损耗高；但提高功率因数需要电容补偿，有成本。工业用户通常要求 PF > 0.9。

**"电缆截面越大越好？" → 不是**。电缆越大，电阻越小，损耗越低；但铜价贵、敷设成本高。需要做经济电流密度分析，选取最优截面。

**"变压器为什么需要分接头？" → 调节电压**。电网电压会随负荷波动，分接头调压可以让副边电压保持在额定范围。通常 ±5%，每档 2.5%。

**"电网频率为什么是 50/60Hz？" → 历史原因 + 经济性平衡**。频率低，变压器/电机体积大；频率高，输电线路感抗大、开关灭弧难。50/60Hz 是经验最优值。
