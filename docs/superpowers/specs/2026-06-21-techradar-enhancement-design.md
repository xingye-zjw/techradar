## 🎨 1. 视觉体验增强

### 页面过渡动画

- 页面切换时添加淡入淡出效果（200ms）
- 路由图节点点击时添加缩放动画

### 悬停效果增强

- 卡片悬停时轻微上浮（`translateY -2px`）+ 阴影增强
- 按钮悬停时颜色渐变过渡

### 加载状态

- 骨架屏加载效果（替代当前的 loading.tsx）
- 数据加载时显示进度指示器

### 反馈动画

- 操作成功时显示绿色勾号动画
- 错误时显示红色抖动效果

## 🔍 2. 全局搜索增强（简化版）

### 跨模块搜索架构

```
搜索输入 → 搜索引擎（fuse.js 增强）
         ↓
    ┌─────────────────────────────────────┐
    │  路由节点  │  情报卡片  │  工具  │  踩坑  │
    └─────────────────────────────────────┘
         ↓
    统一搜索结果页（/search）
```

### 搜索结果展示

- 结果按模块分组：节点、情报、工具、踩坑
- 每个结果显示来源模块标签
- 点击结果直接跳转到对应页面

### 排序规则

- 只按 **相关性** 排序（关键词匹配度）
- 去掉热度统计，保持简单

### 搜索体验

- 输入防抖（300ms）
- 搜索结果高亮匹配关键词
